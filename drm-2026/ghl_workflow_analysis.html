7. Improvement Recommendations

A. Simplification

Recommendation Details
A1. Merge FB Lead Workflows Combine v1 and v2 into one workflow with is-any-of multi-form trigger. Same pipeline, same action — one workflow.
A2. Archive 6 Legacy Pipelines Archive (don't delete) FB Ads, Graphic Design, PPC, SEO, SEO Commercial, Workshops pipelines. Keep DRM 2025-Lead and DRM 2026 Clients.

B. Accuracy & Data Quality

Recommendation Details
B1. Fix {{user.name}} Bug Change to {{contact.name}} in both FB Lead workflows immediately.
B2. Add Monetary Values Set default opportunity values: DRM ~£2,500–5,000/month retainer.

C. Reporting Improvements

Recommendation Details
C1. Opportunity Stage Automation Auto-move opportunities: New Lead → Engaged → Appointment Scheduled → Meeting Completed → Won/Lost
C2. Lead Source Tags Tag contacts: source:facebook-lead, source:google-ads-form
C3. Master Reporting Pipeline Create a single 5-stage pipeline: New → Contacted → Booked → Met → Won/Lost
C4. Email Engagement Tracking Add custom fields: email_sequence_started, emails_opened_count, last_email_opened

D. Notification Improvements

Recommendation Details
D1. No-Show Handling Add: Internal notif → "We missed you" SMS/email → Opp stage "No Show" → Re-engagement sequence
D2. Consolidate Notifications Standardize recipients. Consider Slack/Teams channel for all lead notifications.
D3. Stage Change Notifications Notify on: New lead, Appointment booked, Lead gone cold (14 days), Deal Won
D4. Daily/Weekly Summary Scheduled workflow sending digest: new leads, appointments, pipeline breakdown
